    OK Let's try this

    This is the main page for my apps, this are small utility programs that i make for myself born out of necessity and or curiosity. Feel free to browse, maybe you find something you like.

    Movie Info

    A small front-end to ffprobe (from the ffmpeg package) to show codec information from a movie file, this was mostly an excercise on how to call commandline programs from objective-c but it's nice because you don't have to open the movie in a player to see the bitrate, the codec and other useful info

    iTunes Notifier

    Shows iTunes track info in the Mountain Lion Notification Center so you don't have to switch to iTunes to check the name of the current song.

    Chopcast

    I listen to the Hospital podcast which has really good DnB and comes with chapter information, sometimes i just want to listen to 1 song so i made this tool that will read the podcast and use the chapter info to chop it, pretty much like Max does with a bit mp3 and a cue file.

    Fucking Saved!

    Saves all the images on a 4chan thread to a directory of your choice. Because 4Chan changes their markup a lot i can't guarantee this will always work, but when it does it's sweet!

    Downbooru

    This is an experiment on many things, Website API communication, XML parsing, Application Setting management, ImageKit and other things, but you don't care about that so here is what it does, it allows you to browse a danbooru-type site, search for images with tags and display the previews, the site must have an open API.
